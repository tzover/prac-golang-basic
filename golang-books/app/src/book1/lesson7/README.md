# int型

- int8

```
-128 ~ 127     // 8bit (1byte)
```

- uint8

```
0 ~ 255     // 8bit (1byte)
```

- int16

```
-32,768 ~ 32767     // 16bit (2byte)
```

- uint16

```
0 ~ 65535     // 16bit (2byte)
```

- int32

```
-2,147,483,648 ~ 2,147,483,647     // 32bit (4byte)
```

- uint32

```
0 ~ 4,294,967,295     // 32bit (4byte)
```

- int64

```
-9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807     // 64bit (8byte)
```

- uint64

```
0 ~ 18,446,744,073,709,551,615     // 64bit (4byte)
```
