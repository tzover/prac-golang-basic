```
%T
```

論理値(bool)	%t
符号付き整数(int, int8など)	%d
符号なし整数(uint, uint8など)	%d
浮動小数点数(float64など)	%g
複素数(complex128など)	%g
文字列(string)	%s
チャネル(chan)	%p
ポインタ(pointer)	%p
